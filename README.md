# 🏧 ATM Machine Simulation

A simple ATM Machine Simulation built using HTML and JavaScript. The application allows users to deposit and withdraw money while displaying the updated account balance instantly. It is a beginner-friendly project that demonstrates JavaScript functions, DOM manipulation, conditional statements, and user input handling.

## ✨ Features
- Display current account balance
- Deposit money
- Withdraw money
- Prevent invalid transactions
- Show transaction status messages
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML5
- JavaScript (ES6)

## 📂 Project Structure
```

ATM-Machine/
│── atm.html
│── README.md

```

## 🚀 How to Run
1. Download or clone this repository.
2. Open `atm.html` in any modern web browser.
3. Enter an amount.
4. Click **Deposit** or **Withdraw** to perform transactions.
5. View the updated balance and transaction message.

## 📚 Concepts Used
- DOM Manipulation
- JavaScript Functions
- Conditional Statements
- Variables
- User Input Handling
- Event-Based Programming
